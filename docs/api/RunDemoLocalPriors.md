# Run Demo: Local Priors

**Example script that demonstrates local prior construction and usage.**

This script is provided as a runnable demo for the `HarmonisationFunctions` local priors features.

## Running the demo

```bash
python scripts/run_demo_local_priors.py --help
python scripts/run_demo_local_priors.py --output demo_results.json
```

Inspect the generated `demo_results.json` for example priors and diagnostic outputs. The script is lightweight and intended for demonstration and CI-level smoke tests.
