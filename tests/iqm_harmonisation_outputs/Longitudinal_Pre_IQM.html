Longitudinal Data Diagnostic Report Introduction

2026-05-27 17:06:44,706 — INFO — [SECTION] Longitudinal Data Diagnostic Report Introduction
This report provides diagnostic analyses for longitudinal data collected across multiple batches.

Longitudinal data consist of repeated measurements from the same subjects over time. Such designs require evaluation of measurement stability, batch-related variability, and preservation of biologically meaningful signal.

The following diagnostics are performed:

1. Subject-level variability
   • Subject order consistency (rank preservation across timepoints)
   • Within-subject variability (Coefficient of Variation / Relative Percent Difference)

2. Batch-level variability
   • Additive batch effects (mean shifts; mixed-effects models)
   • Pairwise batch mean differences (post-hoc comparisons)
   • Multiplicative batch effects (variance differences; Fligner–Killeen test)
   • Multivariate batch differences relative to a reference (Mahalanobis distance)

3. Between-subject variability
   • Intra-Class Correlation (ICC; variance decomposition via mixed models)

4. Biological variability
   • Statistical significance of biological covariates
   • Effect sizes (beta coefficients)
   • 95% confidence intervals

Together, these diagnostics assess whether harmonisation reduces unwanted batch effects while preserving meaningful biological and between-subject variation.
